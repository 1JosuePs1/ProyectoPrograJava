Final Project - Soda Management System (Java)
This Java application is a management system for a food outlet ("Soda"), developed as a final project for the programming course using Object-Oriented Programming (OOP).

System Features
Authentication: A secure login module in Soda.java using arrays for user verification.

Ordering Logic: In Cliente.java, users can select items from the menu through graphical dialog boxes.

Billing System: Factura.java handles subtotal calculations, tax (IVA) application, and invoice generation.

Data Management: Uses classes like Producto.java and Proveedor.java to manage stock, pricing, and suppliers.

OOP Principles: The project implements encapsulation (getters/setters), dynamic lists (ArrayList), and exception handling (Try-Catch).
